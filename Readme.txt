

----------------------------------------- SCRAPER AUTOMATION SOFTWARE -----------------------------------------------------------
KEY FEATURES
- using Nightwatchjs and selenium. 
- NO NEED TO CONFIGURE json files or path COMPLETLY DYNAMIC. 
- starts selenium on random open port dynamically.
- no need to conifigure web drivers.
- using firebase to add entries and update it in REAL TIME
- better GUI for user friendly software (npm start and open localhost:8081)
- USING HEADLESS BROWSER SCRIPT PHANTOMJS , NIGHTMAREJS , CASPERJS , SLIMERJS NO RECORDS FOR COOKIES AND SESSION RESQUESTS
  (only implemented in image downloader)
- ITS OPEN SOURCE

2) CHANGE VARIBALES IN GLOBAL.JS FOR EG MONSTEMAIL AND MONSTERPASS
3) nightwatch test/monsterTest.js

1) CLONE THE REPO
2) CHANGE VARIBALES IN GLOBAL.JS FOR EG MONSTEMAIL AND MONSTERPASS
3) nightwatch test/linkedinTest.js
